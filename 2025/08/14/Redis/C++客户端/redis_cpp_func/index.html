C++ 客户端连接

C++ 客户端连接 Redis 的基本步骤如下：

  1. 引入 Redis C++ 客户端库头文件
  2. 创建 Redis 客户端对象
  3. 连接到 Redis 服务器

当然，最重要的是需要安装 Redis C++ 客户端库，如 hiredis 或 redis-plus-plus。若还未安装，请跳转：redis-plus-plus

里面的安装步骤非常详细，按照说明进行即可。
